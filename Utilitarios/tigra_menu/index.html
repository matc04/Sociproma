How to start with Tigra Menu?

The easiest way to configure the menu is using your local copy of the Tigra Menu Builder. >CLICK HERE TO BEGIN<
To configure the script manually follow these steps:

  • Review product documentation
  • Review product demos listed above and select which is the closest to your design idea
  • Copy demo files to separate directory and modify configuration files to tune navigation system for your site
  • Alternatively you can use Online Builder to create your menu visually from one of templates available
  • If you have any problems or questions, check out product forum
  • As you have your menu set up, replace menu.js with its size optimized version (10KB>>4KB) to reduce download time and save network traffic. The file can be found in the directory titled 'compressed'

Here are some suggestions:

  • Keep distribution package unchanged. Extract any files from zip file to separate directory before editing them
  • Modify script configuration step-by-step checking menu look in the browsers. This will help you to rollback last changes if they were erroneous
  • Check out product page periodically. We keep working to bring new versions of the product and more support materials for your convenience
